== Description ==

This plugin allows background images to be set for all blocks.
You can set CSS background-image, background-repeat, background-attachment, background-position-x, background-position-y, and background-size.

== Installation ==

Enter "set background" in the plugin search field.
Once you find the plugin, click "Install Now" to install and activate it.
